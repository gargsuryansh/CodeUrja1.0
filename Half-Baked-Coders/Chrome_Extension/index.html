VeriFact AI Chrome Extension

AI-powered news reliability verification for your browser

About the Extension

VeriFact AI is a Chrome extension that leverages artificial intelligence to verify the reliability of news articles in real-time. It helps users distinguish credible information from potential misinformation by providing reliability scores and detailed analysis.

Key Features

  • Automatic Analysis: Automatically detects and analyzes news articles when you visit supported news sites like BBC.com.
  • Reliability Score: Provides a clear score from 0-100 indicating the article's reliability.
  • Detailed Breakdown: Offers insights on source credibility, fact-checking, and content analysis.
  • Visual Gauge: Easy-to-read gauge visualization of reliability scores.
  • Website Integration: View detailed analysis on our website for comprehensive information.

How to Install

Follow these steps to install the extension for testing:

  1. Download the extension files from this repository
  2. Open Chrome and navigate to chrome://extensions
  3. Enable "Developer mode" using the toggle in the top-right corner
  4. Click "Load unpacked" and select the extension directory
  5. The VeriFact AI extension should now appear in your extensions list
  6. Pin the extension to your toolbar for easy access

Usage

Once installed, the extension works automatically:

  • Visit a supported news website (e.g., BBC.com, NYTimes.com)
  • The extension will automatically analyze the article and show a popup with results
  • View the reliability score and detailed analysis in the popup
  • Click "View on Website" for more comprehensive analysis
  • Use the "Reset Analysis" button to re-analyze the article if needed

Technology Stack

  • Frontend: JavaScript, HTML, CSS
  • Backend: Flask API with AI agent integration
  • AI: Powered by "smol agents" and Groq llama-3.3-70b-versatile model
  • OCR: Gemini for image text extraction
  • Chrome Extensions API: For browser integration
